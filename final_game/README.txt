# game.asm

## Description
This project is an assembly language game developed as part of the CSC314 course. 
Authors: Josh Fritz, Aidan Garrett, Kaley Gann

## How to Play
Navigate the board and enter mystery squares to either teleport, cause an explosion, or collect coins and increase your score! Each coin collected increases your score by 15, and each explosion decreases your score by 5. Teleportation will teleport you to a random location on the board. Have fun!

## Features
- Interactive gameplay
- Score tracking and coin collection
- Teleportation via mystery squares
- Bombs via mystery squares
- Coins via mystery squares
